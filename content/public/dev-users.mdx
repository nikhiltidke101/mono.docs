## dev-users-get

Gets the requested user's information.

### Query Parameters

<Property required={true} deprecated={false} name={"id"} type={"string"}>

User ID of the requested Dev user.

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `404` | 404 Not Found. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

  /**
   * IDs of the Dev User outside the DevRev SOR.

  /**
   * External ref is a mutable unique identifier for a user within the
   * Rev organization from your primary customer record. If none is
