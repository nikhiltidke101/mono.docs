## webhooks-create

Creates a new webhook target.

### Request Body (Optional)



<Property required={false} deprecated={false} name={"event_types"} type={"array of string"}>

The event types that the webhook endpoint will receive.

</Property>

<Property required={false} deprecated={false} name={"secret"} type={"string"}>

The secret to use when verifying webhook events. If provided, the
secret must be between 8 and 32 bytes (inclusive). If not set, a
secret will be automatically generated and provided in the
response.


<span>Format:  `"byte"`</span>

</Property>

<Property required={true} deprecated={false} name={"url"} type={"string"}>

The URL of the webhook endpoint.

</Property>

| Status code | Description |
| ----------- | ----------- |
| `201` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X POST "https://api.devrev.ai/webhooks.create" \
  -d '{
  "event_types": [
    "account_created"
  ],
  "secret": "string",
  "url": "string"
}'
```
