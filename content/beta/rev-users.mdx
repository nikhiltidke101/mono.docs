## rev-users-create

Creates a Rev user for a Rev organization. Rev user can be a customer
or a lead of an organization.


### Request Body (Optional)



<Property required={false} deprecated={false} name={"artifacts"} type={"array of string"}>

The IDs of the artifacts to associate with the Rev user.

<span>Example:  `["ARTIFACT-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"custom_fields"} type={"object"}>

Application-defined custom fields.

<Accordions>

<Accordion title={"Object Type"}>



</Accordion>

</Accordions>

</Property>

<Property required={false} deprecated={false} name={"custom_schema_fragments"} type={"array of string"}>

The schema fragment IDs associated with the Rev user.

<span>Example:  `["don:core:\<partition\>:devo/<dev-org-id>:custom_type_fragment/<custom-type-fragment-id>"]`</span>

</Property>

<Property required={false} deprecated={false} name={"description"} type={"string"}>

Description of the Rev user.

</Property>

<Property required={false} deprecated={false} name={"display_name"} type={"string"}>

The user's display name. The name is non-unique and mutable.


</Property>

<Property required={false} deprecated={false} name={"email"} type={"string"}>

Email address of the Rev user.

</Property>

<Property required={false} deprecated={false} name={"external_ref"} type={"string"}>

External ref is a mutable unique identifier for a user within the
Dev organization from your primary customer record. If none is
available, a good alternative is the email address/phone number
which could uniquely identify the user. If none is specified, a
system-generated identifier will be assigned to the user.


</Property>

<Property required={false} deprecated={false} name={"phone_numbers"} type={"array of string"}>

Phone numbers, in E.164 format, of the Rev user.

</Property>

<Property required={false} deprecated={false} name={"rev_org"} type={"string"}>

The ID of the Rev organization to which the created Rev user is
associated.


<span>Example:  `"REV-AbCdEfGh"`</span>

</Property>

<Property required={false} deprecated={false} name={"tags"} type={"array of object"}>

Tags associated with the Rev user.

The ID of the tag.

<span>Example:  `"TAG-12345"`</span>

</Property>

<Property required={false} deprecated={false} name={"value"} type={"string"}>

The value for the object's association with the tag. If specified,
