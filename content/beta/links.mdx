## links-create

Creates a link between two objects to indicate a relationship.


### Request Body (Optional)



<Property required={true} deprecated={false} name={"link_type"} type={"string"}>

Type of link used to define the relationship.

<span>Value in:  `"developed_with" | "imports" | "is_dependent_on" | "is_duplicate_of" | "is_parent_of" | "is_part_of" | "is_related_to" | "serves"`</span>

</Property>

<Property required={true} deprecated={false} name={"source"} type={"string"}>

The ID of the source object.

</Property>

<Property required={true} deprecated={false} name={"target"} type={"string"}>

The ID of the target object.

</Property>

| Status code | Description |
| ----------- | ----------- |
| `201` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X POST "https://api.devrev.ai/links.create" \
  -d '{
  "link_type": "developed_with",
  "source": "string",
  "target": "string"
}'
```
