## timeline-entries-create

Creates a new entry on an object's timeline.

### Request Body (Optional)



<Property required={false} deprecated={false} name={"collections"} type={"array of string"}>

The collection(s) that the entry belongs to, otherwise if not
provided, then the entry's default collection is used.


</Property>

<Property required={false} deprecated={false} name={"expires_at"} type={"string"}>

If set, then the entry is ephemeral and will be deleted after the
provided time. The minimum value should be at least a minute more
than the current timestamp.


<span>Example:  `"2023-01-01T12:00:00.000Z"`</span>

<span>Format:  `"date-time"`</span>

</Property>

<Property required={false} deprecated={false} name={"labels"} type={"array of string"}>

The labels to be associated with the entry.

<span>Minimum length:  `1`</span>

<span>Maximum length:  `64`</span>

</Property>

<Property required={true} deprecated={false} name={"object"} type={"string"}>

The ID of the object to create the timeline entry for.

<span>Example:  `"PROD-12345"`</span>

</Property>

<Property required={false} deprecated={false} name={"private_to"} type={"array of string"}>

If the visibility of the entry is private, this specifies the users
that the entry is private to. Note the creator is always implicitly
included in this list.


<span>Example:  `["DEVU-12345"]`</span>

</Property>

<Property required={true} deprecated={false} name={"type"} type={"string"}>



<span>Value in:  `"timeline_comment"`</span>

</Property>

<Property required={false} deprecated={false} name={"visibility"} type={"string"}>

The visibility of the entry. If 'private', then the entry is only
visible to the creator, 'internal' is visible with the Dev
organization, 'external' is visible to the Dev organzation and Rev
users, and 'public' is visible to all. If not set, then the default
visibility is 'external'.


<span>Value in:  `"external" | "internal" | "private" | "public"`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `201` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

## timeline-entries-list

Lists the timeline entries for an object.

### Query Parameters

<Property required={true} deprecated={false} name={"object"} type={"string"}>

The ID of the object to list timeline entries for.

The collection(s) to list entries from, otherwise if not provided,

</Property>

The cursor to resume iteration from. If not provided, then iteration
