## service-accounts-get

Gets a service account.

### Query Parameters

<Property required={true} deprecated={false} name={"id"} type={"string"}>

The ID of the requested service account.

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `404` | 404 Not Found. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

```bash title="curl"
curl -X GET "https://api.devrev.ai/service-accounts.get"
```
