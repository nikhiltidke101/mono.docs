Filters for question-answers created by any of the provided users.


<span>Example:  `["DEVU-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"cursor"} type={"string"}>

The cursor to resume iteration from. If not provided, then
iteration starts from the beginning.


</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The maximum number of question-answers to return. The default is
'50'.


<span>Format:  `"int32"`</span>

</Property>

<Property required={false} deprecated={false} name={"mode"} type={"string"}>

The iteration mode to use. If "after", then entries after the provided
cursor will be returned, or if no cursor is provided, then from the
beginning. If "before", then entries before the provided cursor will be
returned, or if no cursor is provided, then from the end. Entries will
always be returned in the specified sort-by order.


<span>Value in:  `"after" | "before"`</span>

</Property>

<Property required={false} deprecated={false} name={"owned_by"} type={"array of string"}>

Filters for question-answers owned by any of the provided users.


<span>Example:  `["DEVU-12345"]`</span>

</Property>
