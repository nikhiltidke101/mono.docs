## search-core

Searches for records based on a given query.

### Query Parameters

<Property required={true} deprecated={false} name={"query"} type={"string"}>

The query string. Search query language:
https://docs.devrev.ai/product/search#fields


</Property>

<Property required={false} deprecated={false} name={"cursor"} type={"string"}>

The cursor from where to begin iteration. Start from beginning if not
provided.


</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The maximum number of items to return in a page. The default is '10'.


<span>Format:  `"int32"`</span>

</Property>

<Property required={false} deprecated={false} name={"namespaces"} type={"array of string"}>

The namespaces to search in.

</Property>

<Property required={false} deprecated={false} name={"sort_by"} type={"string"}>

The property on which to sort the search results. The default is
RELEVANCE.


<span>Value in:  `"created_date" | "modified_date" | "relevance"`</span>

</Property>

<Property required={false} deprecated={false} name={"sort_order"} type={"string"}>

Sorting order. The default is DESCENDING.

<span>Value in:  `"asc" | "desc"`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

curl -X GET "https://api.devrev.ai/search.core"

## search-hybrid

Performs search, using a combination of syntactic and semantic search.


### Query Parameters

<Property required={true} deprecated={false} name={"namespace"} type={"string"}>

The hybrid namespace to search in.

<span>Value in:  `"article" | "conversation" | "issue" | "part" | "question_answer" | "ticket" | "work"`</span>

</Property>

<Property required={true} deprecated={false} name={"query"} type={"string"}>

The query string.

<span>Minimum length:  `1`</span>

<span>Maximum length:  `400`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The maximum number of items to return in a page. The default is '10'.


<span>Minimum:  `0`</span>

<span>Maximum:  `50`</span>

<span>Format:  `"int32"`</span>

</Property>

<Property required={false} deprecated={false} name={"semantic_weight"} type={"number"}>

The weightage for semantic search. Values between 0 and 1 are
accepted.


<span>Minimum:  `0`</span>

<span>Maximum:  `1`</span>

<span>Format:  `"float"`</span>

</Property>

## search-hybrid-post

Performs search, using a combination of syntactic and semantic search.


### Request Body (Optional)



<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The maximum number of items to return in a page. The default is
'10'.


<span>Minimum:  `0`</span>

<span>Maximum:  `50`</span>

<span>Format:  `"int32"`</span>

</Property>

<Property required={true} deprecated={false} name={"namespace"} type={"string"}>

The namespaces for hybrid search.

<span>Value in:  `"article" | "conversation" | "issue" | "part" | "question_answer" | "ticket" | "work"`</span>

</Property>

<Property required={true} deprecated={false} name={"query"} type={"string"}>

The query string.

<span>Minimum length:  `1`</span>

<span>Maximum length:  `400`</span>

</Property>

<Property required={false} deprecated={false} name={"semantic_weight"} type={"number"}>

The weightage for semantic search. Values between 0 and 1 are
accepted.


<span>Minimum:  `0`</span>

<span>Maximum:  `1`</span>

<span>Format:  `"float"`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |
