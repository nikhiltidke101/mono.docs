## dev-users-identities-link

Links an external/secondary identity to the Dev user.

### Request Body (Optional)



<Property required={false} deprecated={false} name={"display_name"} type={"string"}>

Display name of the Dev user in the external source.

</Property>

<Property required={true} deprecated={false} name={"id"} type={"string"}>

Unique ID of the Dev user in the external source.

</Property>

<Property required={true} deprecated={false} name={"issuer"} type={"string"}>

Issuer of the external identity of the Dev user.

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `404` | 404 Not Found. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X POST "https://api.devrev.ai/dev-users.identities.link" \
  -d '{
  "display_name": "string",
  "id": "string",
  "issuer": "string"
}'
```
