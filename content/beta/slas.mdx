## metric-definitions-list

Lists metric definitions matching a filter.

### Query Parameters

<Property required={false} deprecated={false} name={"applies_to_type"} type={"array of string"}>

The type of objects the metric definition applies to.

</Property>

<Property required={false} deprecated={false} name={"cursor"} type={"string"}>

The cursor to resume iteration from. If not provided, then iteration
starts from the beginning.


</Property>

<Property required={false} deprecated={false} name={"include_custom_metrics"} type={"boolean"}>

Whether to include custom metrics in the response. If not set, then
custom metrics are excluded.


</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The maximum number of records to return. The default is '50'.


<span>Format:  `"int32"`</span>

</Property>

<Property required={false} deprecated={false} name={"mode"} type={"string"}>

The iteration mode to use, otherwise if not set, then "after" is
used.


<span>Value in:  `"after" | "before"`</span>

</Property>

<Property required={false} deprecated={false} name={"status"} type={"array of string"}>

The status of the metric definition.

</Property>

<Property required={false} deprecated={false} name={"type"} type={"array of string"}>

The type of metric definitions sought.

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>

<span>Format:  `"int32"`</span>

The iteration mode to use. If "after", then entries after the provided
cursor will be returned, or if no cursor is provided, then from the
beginning. If "before", then entries before the provided cursor will be
returned, or if no cursor is provided, then from the end. Entries will
always be returned in the specified sort-by order.


<span>Value in:  `"after" | "before"`</span>

</Property>
