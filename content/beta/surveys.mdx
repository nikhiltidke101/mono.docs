## surveys-create

Creates a schema for survey, which includes name and description of
schema.


### Request Body (Optional)



<Property required={false} deprecated={false} name={"description"} type={"string"}>

Description about the survey.

</Property>

<Property required={true} deprecated={false} name={"name"} type={"string"}>

The survey's name.

</Property>

<Property required={true} deprecated={false} name={"schema"} type={"array of object"}>

Schema for the survey.

## surveys-responses-list

List survey responses requested by the user.

### Query Parameters

<Property required={false} deprecated={false} name={"created_by"} type={"array of string"}>

<span>Example:  `["DEVU-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"cursor"} type={"string"}>

The cursor to resume iteration from. If not provided, then iteration
starts from the beginning.


</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The maximum number of survey responses to return. If not set, then
the default is '50'.


<span>Format:  `"int32"`</span>

</Property>

<Property required={false} deprecated={false} name={"mode"} type={"string"}>

The iteration mode to use, otherwise if not set, then "after" is
used.


<span>Value in:  `"after" | "before"`</span>

</Property>

<Property required={false} deprecated={false} name={"objects"} type={"array of string"}>

Filters for survey responses created for the objects.

<span>Example:  `["ACC-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"sort_by"} type={"array of string"}>

Fields to sort the survey responses by and the direction to sort
them.


</Property>

<Property required={false} deprecated={false} name={"stages"} type={"array of integer"}>

Filters for survey response stages.

</Property>

<Property required={false} deprecated={false} name={"surveys"} type={"array of string"}>

Filters for survey responses for the provided survey IDs.

</Property>
