## rev-orgs-create

Creates a Rev organization in the authenticated user's Dev
organization.


### Request Body (Optional)



<Property required={false} deprecated={false} name={"account"} type={"string"}>

Account Id to associate with this Rev organization.

<span>Example:  `"ACC-12345"`</span>

</Property>

<Property required={false} deprecated={false} name={"artifacts"} type={"array of string"}>

The IDs of the artifacts to associate with the Rev organization.


<span>Example:  `["ARTIFACT-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"custom_fields"} type={"object"}>

Application-defined custom fields.

<Accordions>

<Accordion title={"Object Type"}>



</Accordion>

</Accordions>

</Property>

<Property required={false} deprecated={false} name={"custom_schema_fragments"} type={"array of string"}>

Schema fragment IDs associated with this Rev organization.


<span>Example:  `["don:core:\<partition\>:devo/<dev-org-id>:custom_type_fragment/<custom-type-fragment-id>"]`</span>

</Property>

<Property required={false} deprecated={false} name={"description"} type={"string"}>

Description of the Rev organization.

</Property>

<Property required={true} deprecated={false} name={"display_name"} type={"string"}>

Name of the Rev organization.

</Property>

<Property required={false} deprecated={false} name={"domain"} type={"string"}>

Company's domain name. Example - 'devrev.ai'.

</Property>

<Property required={false} deprecated={false} name={"environment"} type={"string"}>

The environment of the Org. Defaults to 'production' if not specified.


<span>Value in:  `"production" | "staging" | "test"`</span>

</Property>

<Property required={false} deprecated={false} name={"external_ref"} type={"string"}>

External ref is a custom unique identifier which is a reference to
an unique id for this organization's data in some system of
records.


</Property>

<Property required={false} deprecated={false} name={"tags"} type={"array of object"}>

Tags associated with the Rev organization.

The value for the object's association with the tag. If specified,
the value must be one that's specified in the tag's allowed values.


</Property>
