## engagements-count

Counts the engagement records.

### Query Parameters

<Property required={false} deprecated={false} name={"external_ref"} type={"array of string"}>

</Property>

<Property required={false} deprecated={false} name={"members"} type={"array of string"}>

Filters for engagement of the provided members.

<span>Example:  `["DEVU-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"parent"} type={"array of string"}>

Filters for engagements with the provided parent.

<span>Example:  `["ACC-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"type"} type={"array of string"}>

Filters for engagement of the provided types.

</Property>

## engagements-create

Creates a new engagement record.

### Request Body (Optional)



<Property required={false} deprecated={false} name={"artifacts"} type={"array of string"}>

The IDs of the artifacts to associate with the engagement.


<span>Example:  `["ARTIFACT-12345"]`</span>

</Property>

<Property required={false} deprecated={false} name={"description"} type={"string"}>

The description of the engagement.

</Property>

<Property required={false} deprecated={false} name={"engagement_type"} type={"string"}>

The type of engagement.

<span>Value in:  `"call" | "default" | "email" | "linked_in" | "offline"`</span>

</Property>

<Property required={false} deprecated={false} name={"external_ref"} type={"string"}>

External Reference for the engagement.

</Property>

<Property required={false} deprecated={false} name={"external_url"} type={"string"}>

External URL for the engagement.

</Property>

<Property required={true} deprecated={false} name={"members"} type={"array of string"}>

IDs of the users that were part of the engagement.

<span>Example:  `["DEVU-12345"]`</span>

</Property>

<Property required={true} deprecated={false} name={"parent"} type={"string"}>

The parent object ID in which the engagement was created.
Currently, only accounts and opportunities are supported.


<span>Example:  `"ACC-12345"`</span>

</Property>

<Property required={true} deprecated={false} name={"scheduled_date"} type={"string"}>

The date and time when the engagement was scheduled.

<span>Example:  `"2023-01-01T12:00:00.000Z"`</span>

<span>Format:  `"date-time"`</span>

</Property>

<Property required={false} deprecated={false} name={"tags"} type={"array of object"}>

Tags associated with the engagement.

<Accordions>

<Accordion title={"Object Type"}>

<Property required={true} deprecated={false} name={"id"} type={"string"}>

The ID of the tag.

<span>Example:  `"TAG-12345"`</span>

</Property>

<Property required={false} deprecated={false} name={"value"} type={"string"}>

The value for the object's association with the tag. If specified,
the value must be one that's specified in the tag's allowed values.


</Property>

</Accordion>

</Accordions>

</Property>

<Property required={true} deprecated={false} name={"title"} type={"string"}>

The title of the engagement.

</Property>

| ----------- | ----------- |
| `201` | Success. |
| `400` | 400 Bad Request. |
| `401` | 401 Unauthorized. |
| `403` | 403 Forbidden. |
| `429` | 429 Too Many Requests. |
| `500` | 500 Internal Server Error. |
| `503` | 503 Service Unavailable. |

</APIInfo>
